# Go_Bypass
一个免杀生成器模板，目前可以过国内主流杀毒。
## 捆绑功能 2022-3-22
1. Golang 捆绑文件体积优化 3M 左右
2. 捆绑走HTTP请求
<br />Tip. 项目在用，暂不公开
